# Number-Guessing
Number Guessing is a fun and interactive game that challenges players to guess a randomly generated number within a limited number of attempts. Use logic, strategy, and provided hints to find the correct answer while improving problem-solving and critical-thinking skills.

# Guess the Number — Fun Level Challenge

Guess the Number is an engaging browser-based number guessing game built with Tailwind CSS and vanilla JavaScript. It includes a guided tutorial, progressive levels, power-ups, sound feedback, and animated visuals for a polished gameplay experience.

## Features

- 250+ progressive challenge levels
- Adaptive range difficulty and level modes
- Hot/cold feedback with a visual heat gauge
- Coin-based power-up shop with scanner, shield, freeze, and clue tools
- Win streak tracking and reward coins
- Sound effects using Web Audio API
- Animated background particles and confetti bursts
- Tutorial overlay and interactive onboarding
- Responsive layout with modern game-style UI

## Built With

- HTML
- Tailwind CSS (CDN)
- JavaScript
- Font Awesome icons
- Google Fonts

## Project Structure

- `index.html` — game UI and layout
- `style.css` — styles, animations, and visual polish
- `app.js` — game logic, level system, power-ups, audio, and animations

## Usage

1. Open `index.html` in your browser.
2. Follow the tutorial to learn the rules.
3. Enter a guess and submit it to see hints and progress.
4. Use coins to buy power-ups when you need extra help.
5. Complete levels to earn rewards and advance.

## Notes

- The game runs entirely in the browser; no installation required.
- For the best experience, play in a modern browser with JavaScript enabled.
